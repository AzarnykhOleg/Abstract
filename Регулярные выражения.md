# Базовые правила Python

> **Raw-строки (сырые строки):** Так как регулярные выражения активно используют символ обратного слеша `\`, который в Python также является экранирующим символом, рекомендуется использовать raw-строки, предваряя строку буквой `r`. Это предотвращает двойное экранирование.

```python
import re
# Правильно: с raw-строкой
pattern = r"\d+"
# Эквивалентно без raw-строки (сложнее читать)
pattern = "\\d+"
```

# Таблица синтаксиса регулярных выражений

| Символ/Конструкция |                                     Описание                                      |                    Пример                     |                        
| :----------------: | :-------------------------------------------------------------------------------: | :-------------------------------------------: |
|       **.**        |                  Любой одиночный символ, кроме новой строки `\n`                  |             `a.c` → “abc”, “a-c”              |
|       **^**        |             Начало строки (или начало строки в многострочном режиме)              |   `^Hello` → строки, начинающиеся с “Hello”   |                        
|       **$**        |                                   Конец строки                                    | `world$` → строки, заканчивающиеся на “world” |                       
|       **\d**       |                      Любая цифра (0-9), эквивалентно `[0-9]`                      |                `\d{3}` → “123”                |                     
|       **\D**       |                  Любой нецифровой символ, эквивалентно `[^0-9]`                   |                 `\D+` → “abc”                 |                    
|       **\w**       | Словообразующий символ (буквы, цифры, подчеркивание), эквивалентно `[a-zA-Z0-9_]` |               `\w+` → “word_1”                |              
|       **\W**       |                             Не-словообразующий символ                             |                `\W` → “!”, “ “                | 
|       **\s**       |                Пробельный символ (пробел, табуляция, новая строка)                |      `\s+` → один или несколько пробелов      |   
|       **\S**       |                                Непробельный символ                                |       `\S+` → любое слово без пробелов        |   
|      **[…]**       |                         Набор символов (character class)                          |           `[aeiou]` → любая гласная           |   
|     **[\^…]**      |                                 Отрицание набора                                  |    `[^abc]` → любой символ, кроме a, b, c     |  
|       **\|**       |                           Альтернатива (логическое ИЛИ)                           |                    `cat \| dog` → “cat” или “dog”| 
|      **(…)**       |                        Группировка (захватывающая группа)                         |  `(\d+)-(\d+)` → захватывает числа в группы   |  
|     **(?:…)**      |                              Незахватывающая группа                               |  `(?:http://)` → группировка без сохранения   |   
|     **(?P…)**      |                                Именованная группа                                 |     `(?P<id>\d+)` → группа с именем ‘id’      |   

# Квантификаторы (повторения)

***<span style="color: green">\*</span>*** — **0 или более** повторений предыдущего элемента (жадный).
***<span style="color: green">+</span>*** — **1 или более** повторений (жадный).
***<span style="color: green">?</span>*** — **0 или 1** повторение (необязательный элемент).
***<span style="color: green">{n}</span>*** — ровно **n** повторений.
***<span style="color: green">{n,}</span>*** — **n или более** повторений.
***<span style="color: green">{n,m}</span>*** — от **n до m** повторений.

## Жадность (Greedy vs Non-greedy)

> По умолчанию квантификаторы **жадные** — захватывают максимально возможное количество символов. Для **ленивого** (нежадного) режима добавляется `?` после квантификатора.

| Жадный  |  Ленивый  | Описание                |
| :-----: | :-------: | ----------------------- |
|   `*`   |   `*?`    | 0 или более, минимум    |
|   `+`   |   `+?`    | 1 или более, минимум    |
|   `?`   |   `??`    | 0 или 1, предпочитает 0 |
| `{n,m}` | \`{n,m}?` | от n до m, минимум      |

```python
text = "<div>content</div><div>more</div>"
# Жадный: захватывает всё до последнего </div>
re.search(r'<div>.*</div>', text).group()  # "<div>content</div><div>more</div>"
# Ленивый: захватывает минимум до первого </div>
re.search(r'<div>.*?</div>', text).group() # "<div>content</div>"
```

# Основные функции модуля re

***<span style="color: green">re.search(pattern, string, flags=0)</span>*** — ищет первое совпадение шаблона **в любом месте** строки. Возвращает объект `Match` или `None`.

***<span style="color: green">re.match(pattern, string, flags=0)</span>*** — ищет совпадение **только в начале** строки (эквивалентно `^pattern`).

***<span style="color: green">re.findall(pattern, string, flags=0)</span>*** — возвращает список всех неперекрывающихся совпадений. Если есть группы — возвращает список групп или кортежей групп.

***<span style="color: green">re.finditer(pattern, string, flags=0)</span>*** — возвращает итератор объектов `Match` для всех совпадений (эффективнее для больших текстов).

***<span style="color: green">re.split(pattern, string, maxsplit=0, flags=0)</span>*** — разбивает строку по шаблону, возвращает список.

***<span style="color: green">re.sub(pattern, repl, string, count=0, flags=0)</span>*** — заменяет совпадения на строку `repl` или результат функции. Возвращает новую строку.

***<span style="color: green">re.compile(pattern, flags=0)</span>*** — компилирует шаблон в объект `Pattern` для многократного использования (повышает производительность).

```python
import re
text = "Email: user@example.com, phone: 123-456-7890"
# search
match = re.search(r'\d{3}-\d{3}-\d{4}', text)
if match:
    print(match.group())  # "123-456-7890"
# findall
emails = re.findall(r'\w+@\w+\.\w+', text)
print(emails)
# split
parts = re.split(r',\s*', text)  # ["Email: user@example.com", "phone: 123-456-7890"]
print(parts) 
# sub
censored = re.sub(r'\d', '*', text)  # заменяет все цифры на *
print(censored)
# compile (для многократного использования)
pattern = re.compile(r'\b\w{4}\b')  # слова ровно из 4 букв
words = pattern.findall("This word test")  # ["This", "word", "test"]
```

# Дополнительные возможности

## Якоря (Anchors)

***<span style="color: green">\b</span>*** — граница слова (между `\w` и `\W` или начало/конец строки).
***<span style="color: green">\B</span>*** — не-граница слова.
***<span style="color: green">^</span>*** — начало строки (в режиме MULTILINE — начало каждой строки).
***<span style="color: green">$</span>*** — конец строки.
