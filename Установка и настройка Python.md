● Переходим на официальный сайт Python в раздел &quot;Загрузки&quot;  https://www.python.org/downloads/  ● Выбираем вашу версию операционной системы.  ● Скачиваем установочный дистрибутив  ● Запускаем установку, отвечаем на вопросы и ждём завершения.

***❗❗❗ Если в установщике есть поле &quot;Add Python to PATH&quot;, обязательно  ставим галочку. Это упростить работу с Python из командной строки. ***

python --version

### Создание виртуального окружения

Для начала создайте каталог для нового проекта.

***❗❗❗ Путь от корня  диска до каталога должен быть без пробелов, а названия директорий написаны  латинскими буквами. Допускается символ подчеркивания. ***

После создания каталога перейдите в него.  Далее создаём копию Python внутри рабочего каталога одной из команд (Windows  или Unix)

```python
python -m venv venv
```

- для Windows

```python
python3 -m venv venv
```

- для Linux и MacOS

Остаётся активировать виртуальное окружение. Это финальная команда, которая  отличается в Windows и Unix системах.

```python
venv&#92;Scripts&#92;activate
```

- для Windows

```python
source venv/bin/activate
```

- для Linux и MacOS

Если в начале командной строки вы видите приставку (venv), значит активация  прошла успешно.

для завершения работы внутри виртуального окружения  необходимо выполнить команду

```python
deactivate
```

## Возврат в начало цикла, continue
([29](zotero://open-pdf/library/items/7WNSNKIW?page=29&annotation=MWCUJRQE))

При необходимости работу цикла можно прервать и досрочно вернуться к проверке  условия.

Выведем все чётные числа (как в прошлом примере), кроме тех, которые кратны 12.

```python
num = float(input(&#39;Введите число: &#39;))  STEP = 2  limit = num - STEP  count = -STEP  while count &lt; limit:  count += STEP  if count % 12 == 0:  continue  print(count)
```

> 1. STEP = 2 — добавили константу для движения по чётным и ушли от  &quot;магических чисел&quot;. Теперь изменение условия на &quot;вывод чисел кратных 5&quot;  потребует замены числа в одном месте кода.  2. Ввели переменную limit. Чтобы цикл не выводил лишние числа, больше  введенного num, на каждой проверке цикла while надо вычитать значение  шага. Но шаг — константа. Для экономии ресурсов ПК и ускорения работы  кода логично сделать вычитание один раз перед циклом и сравнивать  значения быстрее, без вычитания в строке while
([29](zotero://open-pdf/library/items/7WNSNKIW?page=29&annotation=GX2MJLRB))

### Действие после цикла, else

Зарезервированное слово else может применятся не только к ветвлениям, но и к  циклам. Для этого else должно быть расположено на том же уровне, т.е. иметь  столько же пробельных отступов, что и начало цикла — while.

Доработаем прошлую программу и дадим 3 попытки на попадание в диапазон.

```python
min_limit = 0  max_limit = 10  count = 3  while count &gt; 0:  print(&#39;Попытка &#39; + str(count))  count -= 1
```

```python
num = float(input(&#39;Введи число между &#39; + str(min_limit) + &#39; и  &#39; + str(max_limit) + &#39;: &#39;))  if num &lt; min_limit or num &gt; max_limit:  print(&#39;Неверно&#39;)  else:  break  else:  print(&#39;Исчерпаны все попытки. ### Цикл по целым числам, он же  арифметический цикл, функция range()
```

Для перебора целых чисел цикл for in используется в связке с функцией range().

```python
num = int(input(&#39;Введите число: &#39;))  for i in range(0, num, 2):  print(i)
```

> Аргументами функции range() должны быть целые числа, int()
([32](zotero://open-pdf/library/items/7WNSNKIW?page=32&annotation=XTH7KRSW))

***❗❗❗ Аргументами функции range() должны быть целые числа, int() ***

`range(stop)`

— перебираем значения от нуля до stop исключительно с шагом один

`range(start, stop)`

— перебираем значения от start включительно до stop  исключительно с шагом один

> range(stop) — перебираем значения от нуля до stop исключительно с шагом один  range(start, stop) — перебираем значения от start включительно до stop  исключительно с шагом один  range(start, stop, step) — перебираем значения от start включительно до stop  исключительно с шагом step.
([32](zotero://open-pdf/library/items/7WNSNKIW?page=32&annotation=4QKJTBJA))

`range(start, stop, step)`

— перебираем значения от start включительно до stop  исключительно с шагом step.

> ● Если значение step отрицательное, перебор будет в сторону уменьшения. 32 ● Если start больше stop при положительном step или наоборот start меньше  stop при отрицательном step, цикл не сработает ни разу. range(10, 5, 2) ничего
([32](zotero://open-pdf/library/items/7WNSNKIW?page=32&annotation=YY5GKDL3))

```python
count = 10  for i in range(count):  for j in range(count):  for k in range(count):  print(i, j, k)
```

### Цикл с нумерацией элементов, функция  enumerate()

enumerate(). Она позволяет добавить  порядковый номер к элементам итерируемой последовательности.

```python
animals = [&#39;cat&#39;, &#39;dog&#39;, &#39;wolf&#39;, &#39;rat&#39;, &#39;dragon&#39;]  for i, animal in enumerate(animals, start=1):  print(i, animal)
```

> ● После for указано две переменные через запятую. В i будет помещаться  порядковый номер. В animal очередное животное из списка.  ● Функция enumerate() получила в качестве первого аргумента список  животных. Второй аргумент — стартовое значение счётчика, т.е. первое  значение, которое попадёт в i.  Если второй аргумент не передать, нумерация начнётся с нуля.
([34](zotero://open-pdf/library/items/7WNSNKIW?page=34&annotation=BN2A6NU9))
