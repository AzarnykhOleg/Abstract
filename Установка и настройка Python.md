### Чтобы гарантированно иметь нужный интерпретатор стоит установить  актуальную версию Python.


● Переходим на официальный сайт Python в раздел &quot;Загрузки&quot;  https://www.python.org/downloads/  
● Выбираем вашу версию операционной системы.  
● Скачиваем установочный дистрибутив  
● Запускаем установку, отвечаем на вопросы и ждём завершения.


***❗❗❗ Если в установщике есть поле &quot;Add Python to PATH&quot;, обязательно  ставим галочку. Это упростить работу с Python из командной строки. ***

Чтобы убедиться в установке интерпретатора откроем командную строку вашей ОС  и введем команду

`python --version`

### Создание виртуального окружения

Для начала создайте каталог для нового проекта.

***❗❗❗ Путь от корня  диска до каталога должен быть без пробелов, а названия директорий написаны  латинскими буквами. Допускается символ подчеркивания. ***

После создания каталога перейдите в него.  Далее создаём копию Python внутри рабочего каталога одной из команд (Windows  или Unix)

```python
python -m venv venv
```

- для Windows

```python
python3 -m venv venv
```

- для Linux и MacOS

Остаётся активировать виртуальное окружение. Это финальная команда, которая  отличается в Windows и Unix системах.

```python
venv&#92;Scripts&#92;activate
```

- для Windows

```python
source venv/bin/activate
```

- для Linux и MacOS

Если в начале командной строки вы видите приставку (venv), значит активация  прошла успешно.

для завершения работы внутри виртуального окружения  необходимо выполнить команду

```python
deactivate
```

### Работа с pip

`Package Installer for Python`— система управления пакетами, которая используется  для установки и управления программными пакетами, написанными на Python.


***❗❗❗ Прежде чем устанавливать дополнения убедитесь, что вы  находитесь в каталоге проекта и активировали виртуальное окружение. ***

### ➢ Команда install

Для установки используем команду install далее указываем название  устанавливаемого дополнения.

```python
pip install requests
```

&quot;Пип&quot; скачает и установит библиотеку requests последней версии в папку venv  виртуального окружения. В основной системе и в других окружениях библиотеки не  будет.

### ➢ Команда freeze

```python
pip freeze
```
Мы получили список всех дополнений внутри нашего виртуального окружения.

### ➢ Файл requirements.txt

В него помещают перечень всех уже  установленных дополнений. Для этого используют команду

```python
pip freeze &gt; requirements.txt
```

Для быстрой установки в новое окружение всего содержимого файла используется  команда

```python
pip install -r requirements.txt
```

### ➢ Отличия командной строки ОС и интерпретатора Python

Выполним команду python (либо python3) чтобы активировать  интерпретатор

Обратите внимание на три треугольные стрелки. Теперь мы находимся в  интерактивном сеансе Python. В нём не будут работать команды ОС, такие как cd,  pip и другие.

### Работа в режиме интерпретатора

каждая строка  кода выполняется сразу после нажатия клавиши Ввод.

если нам нужен последний вывод, можно воспользоваться переменной &quot;_&quot;. Она  хранит именно финальный результат, т.е. обновляется с каждым новым выводом.
