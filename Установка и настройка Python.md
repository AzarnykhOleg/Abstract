# Установить  актуальную версию Python в Windows:
---

● Переходим на официальный сайт Python в раздел [Загрузки](https://www.python.org/downloads/ ) 
● Выбираем вашу версию операционной системы.  
● Скачиваем установочный дистрибутив  
● Запускаем установку, отвечаем на вопросы и ждём завершения.

***❗❗❗ Если в установщике есть поле "Add Python to PATH" обязательно  ставим галочку. Это упростить работу с Python из командной строки. ***

# Установить  актуальную версию Python на Linux:
---

## Узнать текущую установленную версию:
```python
python --version
```
или
```python
python3 --version
```

## Установка требований
```python
sudo apt-get update
```
```python
sudo apt-get install build-essential checkinstall
```
```python
sudo apt-get install libreadline-gplv2-dev libncursesw5-dev libssl-dev libsqlite3-dev tk-dev libgdbm-dev libc6-dev libbz2-dev
```

## Скачивание архива с исходным кодом и его распаковка
```python
cd /usr/src
```
```python
sudo wget https://www.python.org/ftp/python/3.13.1/Python-3.13.1.tar.xz
```
```python
sudo tar -xf Python-3.13.1.tar.xz
```

## Компиляция исходников Python
```python
cd Python-3.13.1
```
```python
sudo ./configure --enable-optimizations
```
```python
sudo make altinstall
```

## Для запуска актуальной версии при запуске интерпретатора
```python
sudo ln -fs /usr/local/bin/python3.13 /usr/bin/python3.13
```

# Создание виртуального окружения
---

#### Для начала создайте каталог для нового проекта:
```python
mkdir new_project
```

***❗❗❗ Путь от корня  диска до каталога должен быть без пробелов, а названия директорий написаны  латинскими буквами. Допускается символ подчеркивания. ***

#### После создания каталога перейдите в него: 
```python
cd new_project
```

#### Далее создаём копию Python внутри рабочего каталога одной из команд:
- для Windows:
```python
python -m venv venv
```
- для Linux и MacOS:
```python
python3 -m venv venv
```

#### Активировать виртуальное окружение:
- для Windows:
```python
venv\Scripts\activate
```
- для Linux и MacOS:
```python
source venv/bin/activate
```

#### Для завершения работы внутри виртуального окружения  необходимо выполнить команду

```python
deactivate
```

# Работа с pip
---

`Package Installer for Python` — система управления пакетами, которая используется  для установки и управления программными пакетами, написанными на Python.

***❗❗❗ Прежде чем устанавливать дополнения убедитесь, что вы  находитесь в каталоге проекта и активировали виртуальное окружение. ***

### ➢ Команда install

Для установки используем команду install далее указываем название  устанавливаемого дополнения:

```python
pip install requests
```

Pip скачает и установит библиотеку `requests` последней версии в папку venv  виртуального окружения. В основной системе и в других окружениях библиотеки не  будет.

### ➢ Команда freeze

```python
pip freeze
```
выводит список всех дополнений внутри нашего виртуального окружения.

### ➢ Файл requirements.txt

В него помещают перечень всех уже  установленных дополнений. Для этого используют команду:

```python
pip freeze > requirements.txt
```

Для быстрой установки в новое окружение всего содержимого файла используется  команда:

```python
pip install -r requirements.txt
```

# Отличия командной строки ОС и интерпретатора Python
---
Выполним команду `python` (либо `python3`) чтобы активировать  интерпретатор.

Обратите внимание на три треугольные стрелки. Теперь мы находимся в  интерактивном сеансе Python. В нём не будут работать команды ОС, такие как `cd`,  `pip` и другие.

#### Работа в режиме интерпретатора

Каждая строка  кода выполняется сразу после нажатия клавиши Ввод.

Если нам нужен последний вывод, можно воспользоваться переменной "_". Она  хранит именно финальный результат, т.е. обновляется с каждым новым выводом.
