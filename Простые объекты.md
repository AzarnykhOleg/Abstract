# [Целые числа, функция int()](zotero://open-pdf/library/items/J46R29EE?page=16&annotation=T74FDR3A)

```python
int(x, base=10)
```

	Первый аргумент — объект, который мы хотим преобразовать в число.

	Второй аргумент указывает на основание  системы счисления. По умолчанию используется десятичная система, её можно не  указывать. Для base допустимы значения от 2 до 36

```python
x = int("42")  
y = int(3.1415)  
z = int("hello", base=30)  
print(x, y, z, sep='\n')
```

	В  Python "резиновый инт". При этом любой объект хранит в  себе "служебную информацию", которая также занимает место в памяти.

```python
import sys


print(sys.getsizeof(10 ** 100))
```

	Символ подчеркивания в качестве разделителя групп цифр:
`num = 7_901_123_456_789`

	Интерпретатор опускает символ подчеркивания. Они нужны лишь для  программиста, читающего код.

# [Функции bin(), oct(), hex()](zotero://open-pdf/library/items/J46R29EE?page=17&annotation=BTD3FNS2)

	Функции для строкового представления чисел в двоичной, восьмеричной и шестнадцатеричной системах счисления

```python
num = 2 ** 16 - 1  
b = bin(num)  
o = oct(num) 
h = hex(num)  
print(b, o, h, sep='\n')
```

	Обратите внимание на приставку в начале числа, состоящую из нуля и латинской  строчной буквы: 
	0b — двоичное, 
	0o — восьмеричное, 
	0x — шестнадцатеричное  представление.

# [Вещественные числа, функция float()](zotero://open-pdf/library/items/J46R29EE?page=18&annotation=MXVIHCCV)

	Числа с плавающей запятой представлены классом `float`.

	`float` является компактным и  быстрым типом данных для математических операций с плавающей запятой, если  нам не требуется высокая точность результата.

# [Логические типы, функция bool()](zotero://open-pdf/library/items/J46R29EE?page=19&annotation=KHXFMDCV)

	

	Функция `bool()` возвращает одно из двух значений `True` и `False`, в  зависимости от входного значения.

	Все числа преобразуются к истине, за исключением нуля. Он считается ложью.

	Любая строка текста также приводится к истине. Ложью является пустая строка (не  путайте с пробелом).

	Коллекции приводятся к  истине, если в них есть элементы. Пустая коллекция — ложь.
