## [Текущий каталог](zotero://open-pdf/library/items/ZICSMZYA?page=17&annotation=WALXULY5)

> Модуль os предоставляет функции для работы с текущей рабочей директорией.

**<span style="color: green">os.getcwd()</span>** — возвращает строку с путем к текущей рабочей директории (current working directory).

**<span style="color: green">os.chdir(path)</span>** — изменяет текущую рабочую директорию на указанный путь.

```python
import os
print(os.getcwd())  # /home/user/projects
os.chdir('/tmp')
print(os.getcwd())  # /tmp
```

## [Создание каталогов](zotero://open-pdf/library/items/ZICSMZYA?page=17&annotation=37XF38L6)

**<span style="color: green">os.mkdir(path[, mode])</span>** — создает одиночный каталог. Вызывает FileExistsError, если каталог существует, и FileNotFoundError, если родительский путь не существует.

**<span style="color: green">os.makedirs(path, exist_ok=False)</span>** — создает каталог и все промежуточные родительские каталоги рекурсивно. Параметр exist_ok=True предотвращает ошибку при существовании каталога [purpleschool.ru](https://purpleschool.ru/knowledge-base/article/rabota-s-katalogami).

```python
import os
os.mkdir('new_folder')  # только один уровень
os.makedirs('parent/child/grandchild', exist_ok=True)  # вложенные
```

## [Удаление каталогов](zotero://open-pdf/library/items/ZICSMZYA?page=18&annotation=ACYWQ4KQ)

**<span style="color: green">os.rmdir(path)</span>** — удаляет пустой каталог. Вызывает OSError, если каталог не пуст или не существует.

**<span style="color: green">os.removedirs(path)</span>** — удаляет каталог и все пустые родительские каталоги в пути.

**<span style="color: green">shutil.rmtree(path)</span>** — (модуль shutil) рекурсивно удаляет каталог и все его содержимое, включая файлы и подкаталоги.

```python
import os
import shutil
os.rmdir('empty_folder')           # только пустой
shutil.rmtree('folder_with_files')  # с содержимым
```

***❗❗❗ Удалить можно лишь пустой каталог. Если внутри удаляемого  каталога есть другие каталоги или файлы, возникнет ошибка OSError. ***

## [Формирование пути](zotero://open-pdf/library/items/ZICSMZYA?page=19&annotation=DAKVMCKY)

**<span style="color: green">os.path.join(path1, path2, …)</span>** — объединяет компоненты пути с использованием разделителя операционной системы. Предпочтительнее конкатенации строк, так как учитывает платформенные различия (/ vs \) [pykili.github.io](https://pykili.github.io/prog/22-file-manipulation).

**<span style="color: green">os.path.abspath(path)</span>** — возвращает абсолютный путь.

**<span style="color: green">os.path.normpath(path)</span>** — нормализует путь, убирая избыточные разделители и ссылки на родительские каталоги.

**<span style="color: green">os.path.expanduser(path)</span>** — заменяет ~ или ~user на домашний каталог пользователя.

**<span style="color: green">os.path.expandvars(path)</span>** — раскрывает переменные окружения вида \(name или \){name}.

```python
import os
path = os.path.join('folder', 'subfolder', 'file.txt')  # folder/subfolder/file.txt
abs_path = os.path.abspath('file.txt')
home = os.path.expanduser('~/documents')  # /home/user/documents
```

## [Чтение данных о каталогах](zotero://open-pdf/library/items/ZICSMZYA?page=19&annotation=34XTXZFS)

**<span style="color: green">os.listdir(path=‘.’)</span>** — возвращает список имен файлов и каталогов в указанном пути. Не рекурсивный [python-school.ru](https://python-school.ru/blog/python/file-system-with-os-module/).

**<span style="color: green">os.scandir(path=‘.’)</span>** — возвращает итератор объектов DirEntry, содержащих информацию о файлах (более эффективен, чем listdir + статистика).

```python
import os
# listdir
entries = os.listdir('.')  # ['file1.txt', 'folder1', ...]
# scandir с дополнительной информацией
with os.scandir('.') as it:
    for entry in it:
        print(entry.name, entry.is_file(), entry.stat().st_size)
```

## [Проверка на директорию, файл и ссылку](zotero://open-pdf/library/items/ZICSMZYA?page=20&annotation=TSIY8T72)

**<span style="color: green">os.path.exists(path)</span>** — существует ли путь (файл, каталог или ссылка).

**<span style="color: green">os.path.isfile(path)</span>** — является ли путь файлом.

**<span style="color: green">os.path.isdir(path)</span>** — является ли путь каталогом.

**<span style="color: green">os.path.islink(path)</span>** — является ли путь символической ссылкой.

**<span style="color: green">os.path.isabs(path)</span>** — является ли путь абсолютным.

```python
import os
path = 'example.txt'
if os.path.exists(path):
    if os.path.isfile(path):
        print('Это файл')
    elif os.path.isdir(path):
        print('Это каталог')
    if os.path.islink(path):
        print('Это символическая ссылка')
```

## [Обход папок через os.walk()](zotero://open-pdf/library/items/ZICSMZYA?page=20&annotation=AWGYBSXM)

**<span style="color: green">os.walk(top, topdown=True, onerror=None, followlinks=False)</span>** — генератор для рекурсивного обхода дерева каталогов. Возвращает кортежи (dirpath, dirnames, filenames) для каждого каталога в дереве [devhops.ru](https://devhops.ru/code/python/basics/os).

- dirpath — путь к текущему каталогу
- dirnames — список подкаталогов в dirpath (можно модифицировать для управления обходом)
- filenames — список файлов в dirpath

```python
import os
for dirpath, dirnames, filenames in os.walk('.'):
    # Пропускаем скрытые каталоги
    dirnames[:] = [d for d in dirnames if not d.startswith('.')]
    
    for filename in filenames:
        full_path = os.path.join(dirpath, filename)
        print(full_path)
```

## [Переименование файлов](zotero://open-pdf/library/items/ZICSMZYA?page=21&annotation=IK7TV7B3)

**<span style="color: green">os.rename(src, dst)</span>** — переименовывает файл или каталог из src в dst. Если dst существует и является файлом, он будет заменен (на Unix). На Windows вызовет ошибку при существовании.

**<span style="color: green">os.renames(old, new)</span>** — переименует файл и создает промежуточные каталоги при необходимости, удаляет пустые исходные каталоги.

```python
import os
os.rename('old_name.txt', 'new_name.txt')
os.renames('file.txt', 'new_folder/new_subfolder/file.txt')  # с созданием путей
```

## [Перемещение файлов](zotero://open-pdf/library/items/ZICSMZYA?page=21&annotation=8M45WKB8)

**<span style="color: green">shutil.move(src, dst)</span>** — перемещает файл или каталог. Если dst существует и является каталогом, перемещает src внутрь dst. Работает между разными файловыми системами (использует копирование при необходимости) [pykili.github.io](https://pykili.github.io/prog/22-file-manipulation).

```python
import shutil
shutil.move('file.txt', 'new_location/')  # в каталог
shutil.move('file.txt', 'renamed.txt')     # переименование с перемещением
shutil.move('folder/', 'new_parent/')      # перемещение каталога
```

## [Копирование файлов](zotero://open-pdf/library/items/ZICSMZYA?page=22&annotation=B59UTC6R)

**<span style="color: green">shutil.copy(src, dst)</span>** — копирует файл. Если dst — каталог, копирует файл в него с сохранением имени. Сохраняет права доступа, но не метаданные (время создания и др.).

**<span style="color: green">shutil.copy2(src, dst)</span>** — копирует файл с сохранением всех метаданных (как cp -p).

**<span style="color: green">shutil.copyfile(src, dst)</span>** — копирует содержимое файла без метаданных. dst должен быть путем к файлу, не каталогом.

**<span style="color: green">shutil.copytree(src, dst)</span>** — рекурсивно копирует каталог src в dst. dst не должен существовать.

```python
import shutil
shutil.copy('file.txt', 'backup/')           # в каталог
shutil.copy2('file.txt', 'backup/file.txt')  # с метаданными
shutil.copyfile('file.txt', 'copy.txt')      # только содержимое
shutil.copytree('project/', 'project_backup/')  # рекурсивно каталог
```
