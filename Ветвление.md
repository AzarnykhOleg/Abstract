## Если, if
([21](zotero://open-pdf/library/items/7WNSNKIW?page=21&annotation=7DWT53HB))

В Python доступны все 6 операций сравнения:

![[_attachments/Установка и настройка Python/image-21-x56-y184.png]]

```python
pwd = &#39;text&#39;  res = input(&#39;Input password: &#39;)  if res == pwd:  print(&#39;Доступ разрешён&#39;)  print(&#39;Но будьте осторожны&#39;)  print(&#39;Работа завершена&#39;)
```

***❗❗❗ рекомендует использовать в качестве отступа 4 пробела.  Программа будет верно работать и с 2 и с 8 отступами, и даже с символом  табуляции (при условии что во всём коде использован единый стиль  отступания). Но программы проверки кода — линтеры будут выдавать  предупреждения, а коллеги по проекту подзатыльники.  Также обратите внимание, что IDE при верной настройке заменяют нажатие  клавиши TAB на клавиатуре на ввод 4 пробелов. ***

## Иначе, else
([22](zotero://open-pdf/library/items/7WNSNKIW?page=22&annotation=MB24H9EA))

```python
pwd = &#39;text&#39;  res = input(&#39;Input password: &#39;)  if res == pwd:  print(&#39;Доступ разрешён&#39;)  print(&#39;Но будьте осторожны&#39;)  else:  print(&#39;Доступ запрещён&#39;)  print(&#39;Работа завершена&#39;)
```

Слово else относится к тому if, с которым находится на одном уровне.

```python
pwd = &#39;text&#39;  res = input(&#39;Input password: &#39;)  if res == pwd:  print(&#39;Доступ разрешён&#39;)  my_math = int(input(&#39;2 + 2 = &#39;))  if 2 + 2 == my_math:  print(&#39;Вы в нормальном мире&#39;)  else:  print(&#39;Но будьте осторожны&#39;)  else:  print(&#39;Доступ запрещён&#39;)  print(&#39;Работа завершена&#39;)
```

## Еще если, elif
([23](zotero://open-pdf/library/items/7WNSNKIW?page=23&annotation=MGW6VP5M))

Для проверки нескольких выражений используется elfi - сокращение от else if.

```python
color = input(&#39;Твой любимый цвет: &#39;)  if color == &#39;красный&#39;:  print(&#39;Любитель яркого&#39;)  elif color == &#39;зелёный&#39;:  print(&#39;Ты не охотник?&#39;)  elif color == &#39;синий&#39;:  print(&#39;Ха, классика!&#39;)  else:  print(&#39;Тебя не понять&#39;)
```

Проверка работает до первого совпадения. После него дальнейший код  пропускается. А если совпадений нет, срабатывает код после else.

## Выбор из вариантов, match и case
([23](zotero://open-pdf/library/items/7WNSNKIW?page=23&annotation=TU2YGFSF))

После match указываем  23 переменную для сравнения. Далее идёт блок из множества case с вариантами  сравнения.

***❗❗❗ Если у вас стоит Python версии 3.9 и ниже, код не будет работать. ***

```python
color = input(&#39;Твой любимый цвет: &#39;)  match color:  case &#39;красный&#39; | &#39;оранжевый&#39;:  print(&#39;Любитель яркого&#39;)  case &#39;зелёный&#39;:  print(&#39;Ты не охотник?&#39;)  case &#39;синий&#39; | &#39;голубой&#39;:  print(&#39;Ха, классика!&#39;)  case _:  print(&#39;Тебя не понять&#39;)
```

Вместо слова else в данной конструкции используется сочетание case _

## Логические конструкции, or, and, not
([24](zotero://open-pdf/library/items/7WNSNKIW?page=24&annotation=8NR54SMR))

В Python доступны три логических оператора:

> ● and — логическое умножение «И»;  ● or — логическое сложение «ИЛИ»;  ● not — логическое отрицание «НЕ».
([24](zotero://open-pdf/library/items/7WNSNKIW?page=24&annotation=T7YABH86))

Логика их работы

Вычислим високосный год в Григорианском  календаре поэтапно:

```python
year = int(input(&#39;Введите год в формате yyyy: &#39;))  if year % 4 != 0:  print(&quot;Обычный&quot;)  elif year % 100 == 0:  if year % 400 == 0:  print(&quot;Високосный&quot;)  else:  print(&quot;Обычный&quot;)  else:  print(&quot;Високосный&quot;)
```

А теперь выберем все случаи, когда год обычный и запишем их в одну строку:

```python
if year % 4 != 0 or year % 100 == 0 and year % 400 != 0:  print(&quot;Обычный&quot;)  else:  print(&quot;Високосный&quot;)
```

### Ленивый if

Если в  логическом выражении есть оператор or и первое значение то есть левое вернуло  истину, дальнейшая проверка не происходит, возвращается True. Если в  логическом выражении есть оператор and и левая половина вернула ложь, то  возвращается False без проверки правой половины выражения.

### Проверка на вхождение, in

Оператор in проверяет вхождение элемента в последовательность.

```python
data = [0, 1, 1, 2, 3, 5, 8, 13, 21]  num = int(input(&#39;Введи число: &#39;))  if num in data:  print(&#39;Леонардо передаёт привет!&#39;)
```

```python
data = [0, 1, 1, 2, 3, 5, 8, 13, 21]  num = int(input(&#39;Введи число: &#39;))  if num not in data:  print(&#39;Леонардо грустит :-(&#39;)
```

### Тернарный оператор

позволяет записать 4  логические строки в одну.

Было:

```python
my_math = int(input(&#39;2 + 2 = &#39;))  if 2 + 2 == my_math:  print(&#39;Верно!&#39;)  else:  print(&#39;Вы уверены?&#39;)
```

Стало:

```python
my_math = int(input(&#39;2 + 2 = &#39;))  print(&#39;Верно!&#39; if 2 + 2 == my_math else &#39;Вы уверены?&#39;)
```

Слева от if записываем выражение для истины, справа от else результат для лжи.

***❗❗❗ PEP-8! требует, чтобы длина строки кода не превышала 80 символов.  Современные мониторы с широкими экранами и высоким разрешением с  легкостью вмещают больше символов. Поэтому ограничение было  пересмотрено и увеличено до 120 символов. Более длинные строки кода в  Python недопустимы, т.к. снижают его читаемость. ***
