# [Множества {set} и ({frozenset})](zotero://open-pdf/library/items/PKBGH6XX?page=33&annotation=WP3FWGK5)

	множество (англ. set) - неупорядоченная изменяемая (mutable) коллекция уникальных хешируемых элементов. Автоматически удаляет дубликаты при создании и добавлении. Поддерживает стандартные математические операции: объединение, пересечение, разность, симметрическую разность.

	frozenset (англ frozen set) - неизменяемая (immutable, hashable) версия множества. Содержит те же элементы и поддерживает те же операции чтения, что и set, но не позволяет добавлять или удалять элементы после создания. Может использоваться в качестве ключа словаря или элемента другого множества.

## Основные свойства множеств

> - ***неупорядоченность*** - элементы не имеют индексов и фиксированной позиции; начиная с Python 3.7 порядок вставки сохраняется при итерации (как у словарей), но индексация не поддерживается
> - ***изменяемость*** (для set) - возможность добавлять и удалять элементы после создания; frozenset неизменяем
> - ***уникальность*** - каждый элемент может встречаться только один раз; дубликаты автоматически отбрасываются
> - ***элементы должны быть хешируемыми (hashable)*** - неизменяемыми: числа, строки, кортежи, frozenset; списки, словари и обычные множества не могут быть элементами множества
> - ***итерабельность*** - поддержка цикла for, распаковки и других операций итерации


## Способы создания множеств

> ```
> Пустое множество и множество с элементами (литерал {elem})
> ```

```python
set_1 = set()
set_2 = {1, 2, 3, 3, 3}  # дубликаты удаляются автоматически
print(set_2)
```

> ```
> Важное различие: пустые фигурные скобки {} создают словарь, а не множество!
> ```

```python
empty_dict = {}
empty_set = set()
print(type(empty_dict))  # <class 'dict'>
print(type(empty_set))   # <class 'set'>
```

> ```
> Создание из итерируемого объекта (список, строка, кортеж, диапазон)
> ```

```python
set_3 = set([1, 2, 3, 3])
set_4 = set("hello")     # {'h', 'e', 'l', 'o'} - буквы без повторов
set_5 = set(range(5))    # {0, 1, 2, 3, 4}
print(set_4)
```

> ```
> Неизменяемое множество frozenset
> ```

```python
frozenset_1 = frozenset()
frozenset_2 = frozenset([1, 2, 3])
frozenset_3 = frozenset("hello")
print(frozenset_2)
```

> ```
> Генератор множества (set comprehension)
> ```

```python
squares = {x**2 for x in range(10)}
evens = {x for x in range(10) if x % 2 == 0}
print(evens)
```

## Базовые операции с элементами

> ```
> Добавление и изменение "одного" элемента
> ```

```python
s = {1, 2, 3}
s.add(4)        # добавление одного элемента
s.add(4)        # повторное добавление игнорируется (дубликатов нет)
print(s)
```

> ```
> Удаление элементов (различные способы)
> ```

```python
s = {1, 2, 3, 4, 5}
# remove - вызывает KeyError, если элемент отсутствует
s.remove(4)
# discard - не вызывает ошибку при отсутствии элемента
s.discard(10)   # спокойно игнорирует отсутствие
# pop - удаляет и возвращает произвольный элемент (в реализации CPython - последний добавленный)
elem = s.pop()
# clear - удаление всех элементов
s.clear()
print(s)
```

> ```
> Проверка наличия элемента (оператор in)
> ```

```python
s = {1, 2, 3}
exists = 2 in s      # True
missing = 5 not in s # True
print(exists)
```

> ```
> Количество элементов
> ```

```python
s = {1, 2, 3}
count = len(s)
print(count)
```

## Методы множеств и операции над множествами

***<span style="color: green">set.union(other, …)</span>*** - возвращает новое множество, содержащее все элементы из текущего и other (объединение). Эквивалент оператора |.

***<span style="color: green">set.intersection(other, …)</span>*** - возвращает пересечение множеств (общие элементы). Эквивалент оператора &.

***<span style="color: green">set.difference(other, …)</span>*** - возвращает разность (элементы из set, которых нет в other). Эквивалент оператора -.

***<span style="color: green">set.symmetric_difference(other)</span>*** - возвращает симметрическую разность (элементы, входящие ровно в одно из множеств). Эквивалент оператора ^.

***<span style="color: green">set.issubset(other)</span>*** - возвращает True, если set является подмножеством other (все элементы set содержатся в other). Эквивалент оператора <=.

***<span style="color: green">set.issuperset(other)</span>*** - возвращает True, если set является надмножеством other (все элементы other содержатся в set). Эквивалент оператора >=.

***<span style="color: green">set.isdisjoint(other)</span>*** - возвращает True, если множества не имеют общих элементов (пересечение пусто).

***<span style="color: green">set.add(elem)</span>*** - добавляет элемент elem в множество; если элемент уже существует, ничего не делает.

***<span style="color: green">set.remove(elem)</span>*** - удаляет элемент elem из множества; вызывает KeyError, если элемент отсутствует.

***<span style="color: green">set.discard(elem)</span>*** - удаляет элемент elem из множества, если он присутствует; не вызывает ошибку при отсутствии элемента.

***<span style="color: green">set.pop()</span>*** - удаляет и возвращает произвольный элемент из множества. Вызывает KeyError, если множество пусто. В CPython удаляет последний добавленный элемент (LIFO), но это деталь реализации.

***<span style="color: green">set.clear()</span>*** - удаляет все элементы из множества, оставляя пустое множество set().

***<span style="color: green">set.copy()</span>*** - возвращает поверхностную (мелкую) копию множества (shallow copy).

***<span style="color: green">set.update(other, …)</span>*** - обновляет множество, добавляя все элементы из other (и других итерируемых). Эквивалент оператора |=. Метод ничего не возвращает (None).

***<span style="color: green">set.intersection_update(other, …)</span>*** - оставляет в множестве только элементы, присутствующие в other (пересечение “на месте”). Эквивалент оператора &=.

***<span style="color: green">set.difference_update(other, …)</span>*** - удаляет из множества все элементы, присутствующие в other. Эквивалент оператора -=.

***<span style="color: green">set.symmetric_difference_update(other)</span>*** - сохраняет в множестве только элементы, входящие ровно в одно из множеств (текущее или other). Эквивалент оператора ^=.

## Операторы множеств (краткая запись)

> ```
> Математические операторы для множеств
> ```
> 
> ```python
> a = {1, 2, 3, 4}
> b = {3, 4, 5, 6}
> # Объединение (union)
> print(a | b)           # {1, 2, 3, 4, 5, 6}
> # Пересечение (intersection)
> print(a & b)           # {3, 4}
> # Разность (difference)
> print(a - b)           # {1, 2}
> # Симметрическая разность
> print(a ^ b)           # {1, 2, 5, 6}
> ```
> 
> ```
> Операторы сравнения (проверка отношений)
> ```
> 
> ```python
> a = {1, 2}
> b = {1, 2, 3}
> print(a <= b)          # True (a - подмножество b)
> print(b >= a)          # True (b - надмножество a)
> print(a < b)           # True (строгое подмножество)
> print(a > b)           # False (строгое надмножество)
> ```
> 
> ```
> Операторы обновления (in-place) для изменяемых множеств
> ```
> 
> ```python
> a = {1, 2}
> b = {2, 3}
> a |= b                 # объединение с присваиванием (a.update(b))
> print(a)               # {1, 2, 3}
> a &= {2, 3, 4}         # пересечение с присваиванием
> print(a)               # {2, 3}
> a -= {3}               # разность с присваиванием
> print(a)               # {2}
> a ^= {1, 2, 3}         # симметрическая разность с присваиванием
> print(a)               # {1, 3}
> ```

## Перебор множества (итерация)

```python
s = {"apple", "banana", "cherry"}
# Простой перебор элементов (порядок произвольный, но стабилен в одном запуске)
print("Элементы множества:")
for item in s:
    print(item)
# Безопасное удаление во время итерации (создаем копию)
print("Удаление 'banana':")
for item in list(s):
    if item == "banana":
        s.remove(item)
print(s)
```

> 
> ```

> ```python

## Особенности frozenset

> ```
> Создание и использование в качестве ключей словаря
> ```
> 
> ```python
> # frozenset является hashable и может быть ключом словаря
> cache = {
>     frozenset({1, 2, 3}): "result A",
>     frozenset({4, 5}): "result B"
> }
> key = frozenset([3, 2, 1])  # порядок не важен для множества
> print(cache[key])           # "result A"
> ```
> 
> ```
> Вложенность множеств (frozenset внутри set)
> ```
> 
> ```python
> # Обычные множества не могут содержать другие множества
> inner_set = frozenset({1, 2})
> outer_set = {inner_set, frozenset({3, 4})}
> print(outer_set)
> ```
> 
> ```
> Операции с frozenset (все неизменяющие методы доступны)
> ```
> 
> ```python
> fs = frozenset([1, 2, 3, 4])
> print(fs.union({5, 6}))           # frozenset({1, 2, 3, 4, 5, 6})
> print(fs.intersection({2, 3, 5}))  # frozenset({2, 3})
> # Изменяющие методы отсутствуют:
> # fs.add(5)      # AttributeError
> # fs.remove(1)   # AttributeError
> ```

## Практические замечания

> ```
> Удаление дубликатов из списка через преобразование в множество (с потерей порядка)
> ```
> 
> ```python
> original = [1, 2, 2, 3, 4, 4, 5]
> unique = list(set(original))
> print(unique)
> ```
> 
> ```
> Сохранение порядка при удалении дубликатов (Python 3.7+)
> ```
> 
> ```python
> original = [1, 2, 2, 3, 4, 4, 5]
> unique_ordered = list(dict.fromkeys(original))  # dict сохраняет порядок
> print(unique_ordered)
> ```
> 
> ```
> Проверка на общие элементы между списками через множества
> ```
> 
> ```python
> list_a = [1, 2, 3, 4]
> list_b = [3, 4, 5, 6]
> if set(list_a) & set(list_b):
>     print("Есть общие элементы:", set(list_a) & set(list_b))
> ```
