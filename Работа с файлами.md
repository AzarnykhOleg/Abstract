# [Файлы](zotero://open-pdf/library/items/ZICSMZYA?page=4&annotation=SKAPB4IL)

## [Функция open()](zotero://open-pdf/library/items/ZICSMZYA?page=5&annotation=B2323Y4N)

> Функция открывает файл и возвращает файловый объект. Обязательные параметры: путь к файлу и режим доступа.

```python
file = open('data.txt', 'r', encoding='utf-8')
```

***<span style="color: green">open(file, mode=‘r’, buffering=-1, encoding=None, errors=None, newline=None, closefd=True, opener=None)</span>*** — встроенная функция для открытия файла.

***❗❗❗ Кодировка UTF-8 является современным стандартом для  хранения и передачи текстовой информации. Если вы явно не планируете  5 работать с другой кодировкой, всегда указывайте encoding=&#39;utf-8&#39; при открытии  текстовых файлов. Это обеспечит переносимость вашего кода между  различными платформами. ***

## [Режимы работы с файлами](zotero://open-pdf/library/items/ZICSMZYA?page=6&annotation=ISTYUH6T)

|Режим|Описание|
|---|---|
|**‘r’**|Чтение (read). Режим по умолчанию. Файл должен существовать.|
|**‘w’**|Запись (write). Создает новый файл или перезаписывает существующий (обнуляет содержимое).|
|**‘x’**|Эксклюзивное создание. Создает файл, если он не существует. Если файл существует — вызывает FileExistsError.|
|**‘a’**|Добавление (append). Запись в конец файла. Создает файл, если не существует.|
|**‘b’**|Бинарный режим (binary). Для работы с байтами (изображения, аудио). Комбинируется: ‘rb’, ‘wb’.|
|**’t’**|Текстовый режим (text). Режим по умолчанию. Комбинируется с другими: ‘rt’, ‘wt’.|
|**’+’**|Чтение и запись (update). Открывает файл для обоих режимов: ‘r+’, ‘w+’, ‘a+’.|

> Режимы можно комбинировать: ‘rb’ — чтение бинарного, ‘w+’ — запись с возможностью чтения, ‘a+’ — добавление с чтением.

```python
# Текстовые режимы
f1 = open('text.txt', 'r', encoding='utf-8')      # только чтение
f2 = open('text.txt', 'w', encoding='utf-8')      # перезапись
f3 = open('text.txt', 'a', encoding='utf-8')      # добавление в конец
f4 = open('text.txt', 'r+', encoding='utf-8')     # чтение и запись
# Бинарные режимы
f5 = open('image.jpg', 'rb')  # чтение бинарного
f6 = open('data.bin', 'wb')   # запись бинарного
```

***❗❗❗ При открытии файла в режиме “b” информация предаётся в виде  байт. Указывать кодировку encoding при этом режиме не нужно, т.к. данные не  декодируются. ***

## [Метод close()](zotero://open-pdf/library/items/ZICSMZYA?page=6&annotation=GRV4PVN2)

***<span style="color: green">ile.close()</span>*** — метод закрывает файл, освобождая системные ресурсы. После закрытия операции чтения/записи вызывают ValueError.

```python
f = open('file.txt', 'r')
data = f.read()
f.close()  # обязательное закрытие
```

> Незакрытые файлы могут привести к утечке ресурсов и потере данных (из-за буферизации записи). Для гарантированного закрытия используйте менеджер контекста with.

***❗❗❗ Если в коде отсутствует метод close(), то даже при успешном  завершении программы не гарантируется сохранение всех данных в файле. ***

## [Менеджер контекста with open](zotero://open-pdf/library/items/ZICSMZYA?page=8&annotation=5SUL3LXU)

> Конструкция with автоматически управляет открытием и закрытием файла, даже при возникновении исключений. Рекомендуемый способ работы с файлами.

```python
# Базовый синтаксис
with open('file.txt', 'r', encoding='utf-8') as f:
    data = f.read()
    # файл автоматически закрывается при выходе из блока
# Эквивалент с try-finally
f = open('file.txt', 'r')
try:
    data = f.read()
finally:
    f.close()
```

## Работа с несколькими файлами

> Для работы с несколькими файлами используются вложенные менеджеры контекста или несколько файловых объектов в одном with.

```python
# Последовательная обработка
with open('input.txt', 'r', encoding='utf-8') as infile:
    data = infile.read()
with open('output.txt', 'w', encoding='utf-8') as outfile:
    outfile.write(data)
# Одновременная работа (копирование построчно)
with open('source.txt', 'r', encoding='utf-8') as src, \
     open('dest.txt', 'w', encoding='utf-8') as dst:
    for line in src:
        dst.write(line)
# Эквивалентная запись без обратного слеша
with (open('a.txt', 'r') as f1,
      open('b.txt', 'r') as f2):
    print(f1.read(), f2.read())
```

# Чтение файла

## [Чтение в список](zotero://open-pdf/library/items/ZICSMZYA?page=9&annotation=IXWSAMJQ)

***<span style="color: green">file.readlines(hint=-1)</span>*** — читает все строки и возвращает список строк. hint ограничивает примерный размер прочитанных байт.

> Для преобразования файла в список можно также использовать list(f) или list comprehension с удалением \n.

```python
# Метод readlines()
with open('file.txt', 'r', encoding='utf-8') as f:
    lines = f.readlines()  # ['первая строка\n', 'вторая\n']
# Конструктор list()
with open('file.txt', 'r', encoding='utf-8') as f:
    lines = list(f)  # тот же результат
# List comprehension без символов новой строки
with open('file.txt', 'r', encoding='utf-8') as f:
    lines = [line.strip() for line in f]
```

## [Чтение всего файла методом read](zotero://open-pdf/library/items/ZICSMZYA?page=9&annotation=BGW43XM2)

***<span style="color: green">file.read(size=-1)</span>*** — читает и возвращает содержимое файла. При size=-1 или None читает весь файл. При указанном size — не более size байт (для бинарных) или символов (для текста).

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    content = f.read()  # весь файл в одну строку
    print(content)
```

## Чтение блоками (чанками)

> Для больших файлов чтение целиком может не поместиться в память. Используйте цикл с указанием размера блока.

```python
# Чтение блоками по 1024 байта
with open('large_file.txt', 'r', encoding='utf-8') as f:
    while True:
        chunk = f.read(1024)  # читаем 1024 символа
        if not chunk:
            break
        print(chunk, end='')
# Чтение бинарного файла блоками
with open('image.jpg', 'rb') as f:
    while chunk := f.read(4096):  # walrus operator (Python 3.8+)
        print(len(chunk))
```

## [Построчное чтение методом readline](zotero://open-pdf/library/items/ZICSMZYA?page=10&annotation=LLKXHGVC)

***<span style="color: green">ile.readline(size=-1)</span>*** — читает и возвращает одну строку (включая символ \n). При указанном size — не более size байт, но может прочитать меньше, если встретит конец строки.

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    first_line = f.readline()    # первая строка
    second_line = f.readline()   # вторая строка
    print(first_line.strip())    # strip() удаляет \n
```

## [Итерация по файлу циклом for](zotero://open-pdf/library/items/ZICSMZYA?page=11&annotation=Q6LRJ4AE)

> Файловый объект является итератором строк. Цикл for читает файл построчно, не загружая его целиком в память.

```python
with open('file.txt', 'r', encoding='utf-8') as f:
    for line in f:  # аналогично for line in f.readlines()
        print(line.strip())
```

```python
with open(&#39;text_data.txt&#39;, &#39;r&#39;, encoding=&#39;utf-8&#39;) as f:  for line in f:  print(line, end=&#39;&#39;)
```

***❗❗❗ Символ переноса строки сохранился в конце каждой строки.  Если вам необходимо обработать строку без переносов, можно использовать  срезы line[:-1] или метод замены line.replace(&#39;&#92;n&#39;, &#39;&#39;) ***

# [Запись и добавление в файл](zotero://open-pdf/library/items/ZICSMZYA?page=11&annotation=AB2RL5FC)

## [Запись методом write](zotero://open-pdf/library/items/ZICSMZYA?page=12&annotation=H4PH29YR)

***<span style="color: green">file.write(string)</span>*** — записывает строку в файл. Возвращает количество записанных символов. Для текстовых файлов требуется строка (str), для бинарных — байты (bytes/bytearray). Не добавляет автоматически перенос строки.

```python
with open('output.txt', 'w', encoding='utf-8') as f:
    f.write('Первая строка\n')
    f.write('Вторая строка\n')
    chars_written = f.write('Третья')
    print(f'Записано символов: {chars_written}')
```

## [Запись методом writelines](zotero://open-pdf/library/items/ZICSMZYA?page=12&annotation=SY36WQKT)

***<span style="color: green">file.writelines(lines)</span>*** — записывает итерируемый объект строк в файл. Не добавляет разделители между строками. Эквивалентен вызову write() для каждой строки.

```python
lines = ['Первая строка', 'Вторая строка', 'Третья строка']
# Неправильно: результат будет в одну строку без разделителей
with open('out.txt', 'w', encoding='utf-8') as f:
    f.writelines(lines)  # Файл содержит: Первая строкаВторая строка...
# Правильно 1: добавляем \n к каждой строке
with open('out.txt', 'w', encoding='utf-8') as f:
    f.writelines(line + '\n' for line in lines)
# Правильно 2: используем join() для создания одной большой строки
with open('out.txt', 'w', encoding='utf-8') as f:
    f.write('\n'.join(lines))  # writelines здесь не нужен
```

## [Перенаправление вывода:print в файл](zotero://open-pdf/library/items/ZICSMZYA?page=13&annotation=N7AV2XT9)

> Функция print() позволяет направить вывод напрямую в файловый объект через именованный параметр file, что удобно для логирования и автоматического преобразования объектов в строки. В отличие от write(), print() автоматически добавляет пробелы между аргументами (параметр sep) и перевод строки в конце (параметр end), если не указано иное.

***print(*objects, sep=’ ‘, end=’\n’, file=file_obj, flush=False)*** — выводит объекты в указанный file_obj. Автоматически вызывает str() для объектов.

```python
with open('log.txt', 'a', encoding='utf-8') as f:
    # Автоматическое преобразование и добавление \n
    print('Ошибка в модуле', 404, 'время:', '12:00', file=f, sep=' | ')
    
    # Без перевода строки (end='')
    print('Загрузка: ', end='', file=f)
    print('50%', file=f)
    
    # Перенаправление stderr (для отладки)
    import sys
    print('Сообщение об ошибке', file=sys.stderr)
```

# [Методы перемещения в файле](zotero://open-pdf/library/items/ZICSMZYA?page=14&annotation=HE7NZIVR)

	Для файлов, открытых в режимах, поддерживающих чтение и запись одновременно (‘r+’, ‘w+’, ‘a+’, ‘rb+’), доступна навигация по содержимому через указатель текущей позиции. Это позволяет читать и перезаписывать данные в произвольном месте файла, а не только последовательно.

## [Метод tell](zotero://open-pdf/library/items/ZICSMZYA?page=14&annotation=6C3UK4NY)

***<span style="color: green">ile.tell()</span>*** — возвращает текущую позицию указателя файла в виде целого числа (количество байт от начала файла). В текстовом режиме значение зависит от кодировки и может отличаться от количества символов; результат tell() можно безопасно использовать только для seek() в этом же файле.

```python
text = ['Lorem ipsum dolor sit amet, consectetur adipisicing  elit.',  'Consequatur debitis explicabo laboriosam sint suscipit  temporibus veniam?',  'Accusantium alias amet fugit iste neque non odit quia  saepe totam velit?', ]  
with open('new_data.txt', 'w', encoding='utf-8') as f:  
	print(f.tell())  
	for line in text:  
		f.write(f'{line}\n')  
		print(f.tell())  
	print(f.tell())  
print(f.tell()) # ValueError: I/O operation on closed file.
```

## [Метод seek](zotero://open-pdf/library/items/ZICSMZYA?page=14&annotation=KTHIATIT)

***<span style="color: green">file.seek(offset, whence=0)</span>*** — изменяет текущую позицию на offset байт относительно точки отсчета whence. Возвращает новую абсолютную позицию.

Параметр whence принимает значения:  
- 0 (os.SEEK_SET) — от начала файла (по умолчанию), offset должен быть >= 0  
- 1 (os.SEEK_CUR) — от текущей позиции, offset может быть отрицательным  
- 2 (os.SEEK_END) — от конца файла, offset обычно отрицательный или 0

**Ограничение для текстовых файлов:** в текстовом режиме (без ‘b’) допускается только whence=0 или whence=2 с offset=0, либо whence=0 с offset, полученным от file.tell(). Произвольное смещение в текстовом режиме может нарушить декодирование многобайтовых символов. Для свободной навигации используйте бинарный режим.

```python
last = before = 0  
text = ['Lorem ipsum dolor sit amet, consectetur adipisicing  elit.',  'Consequatur debitis explicabo laboriosam sint suscipit  temporibus veniam?',  'Accusantium alias amet fugit iste neque non odit quia  saepe totam velit?', ]  
with open('new_data.txt', 'r+', encoding='utf-8') as f:  
	while line := f.readline():  
		last, before = f.tell(), last  
	print(f'{last = }, {before = }')  
	print(f'{f.seek(before, 0) = }')  
	f.write('\n'.join(text))
```

## [Метод truncate](zotero://open-pdf/library/items/ZICSMZYA?page=15&annotation=5I65C4WA)

***<span style="color: green">file.truncate(size=None)</span>*** — изменяет размер файла до size байт. Если size не указан, используется текущая позиция (tell()). При уменьшении размера — обрезает конец файла. При увеличении (редко используется) — дополняет нулями (b’\x00’) или зависит от ОС. Требует открытия файла в режиме, допускающем запись.

```python
import os
# Пример 1: Определение размера файла
with open('data.bin', 'rb') as f:
    f.seek(0, os.SEEK_END)  # перемещаемся в конец
    file_size = f.tell()    # получаем позицию = размер
    print(f"Размер: {file_size} байт")
# Пример 2: Модификация файла в произвольной позиции (бинарный режим)
with open('data.bin', 'r+b') as f:  # чтение + запись, бинарный
    f.seek(5)           # переход к 6-му байту
    byte = f.read(1)    # читаем 1 байт
    f.seek(5)           # возвращаемся назад
    f.write(b'X')       # перезаписываем байт
    
# Пример 3: Обрезка файла (удаление данных с конца)
with open('log.txt', 'r+', encoding='utf-8') as f:
    f.seek(100)         # оставляем первые 100 байт
    f.truncate()        # удаляем всё, что после позиции 100
    
# Пример 4: Запись в середину файла без очистки (текстовый режим, осторожно)
with open('config.txt', 'r+', encoding='utf-8') as f:
    content = f.read()
    f.seek(0)  # только возврат к началу безопасен
    f.write(content.replace('old', 'new'))
```
