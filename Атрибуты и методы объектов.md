# [Атрибуты объекта](zotero://open-pdf/library/items/J46R29EE?page=12&annotation=XGYB8PFU)

	Атрибуты — это переменные, конкретные характеристики объекта, такие как цвет  поля или имя пользователя.
	Атрибут хранит информацию о состоянии  объекта. Для обращения к атрибуту объекта в Python нужно после его имени  поставить точку и далее указать название атрибута.

```python
print("Hello world!".__doc__)
print(str.__doc__)
```

	Мы обратились к атрибуту, в названии которого использованы два подчёркивания до имени и два после.
	Такие переменные называют магическими или дандер. Магическим  может быть как атрибут, хранящий какие-то данные, так и метод.

# [Методы объекта](zotero://open-pdf/library/items/J46R29EE?page=13&annotation=SSH8PAFJ)

	Методы — это функции, которые описаны внутри объекта или класса. Они относятся  к определенному объекту и позволяют взаимодействовать с ними или другими  частями кода. По сути метод — это функция класса.

	В Python для обращения к методу используется точечная нотация, как и с  атрибутами. Отличии в том, что после имени метода ставятся круглые скобки. А  если метод ожидает получить данные на вход, они указываются как аргументы в  скобках.

```python
print("Hello world!".upper())  
print("Hello world!".count("l"))
```

# [Функция dir(object)](zotero://open-pdf/library/items/J46R29EE?page=14&annotation=LUXSM2K6)

	Один из способов проверки наличия атрибутов и методов у объекта - передача его  функции dir() в качестве аргумента.

```python
print(dir("Hello world!"))
```

# [Функция help(object)](zotero://open-pdf/library/items/J46R29EE?page=14&annotation=YQ7JFJCT)

	Встроенная функция help() выводит подсказку об объекте, который передается в  качестве аргумента.

	Функция help() без аргументов запускает интерактивный режим. В нём  можно указывать имена зарезервированных слов, встроенных функций,  модулей и получать справочную информацию.

```python
help()
```

	Для выхода из режима справки используйте команду quit.

	Если передать в функцию имя класса, получим подробное описание его  работы.

```python
help(str)
```
