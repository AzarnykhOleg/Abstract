# [Аннотация типов](zotero://open-pdf/library/items/J46R29EE?page=10&annotation=MEGCIV5K)

	Аннотация  типов упрощает отладку кода: IDE может подсказать возможные ошибки, если  программист присваивает переменной новый, другой тип данных.

```python
a: int = 42  
b: float = 3,14  
a = a / b
```

	После имени переменной ставим двоеточие и через пробел указываем тип данных.

```python
def my_func(data: list[int, float]) -> float: 
	res = sum(data) / len(data) 
	return res  


print(my_func([2, 5.5, 15, 8.0, 13.74]))
```

	Аннотация подсказывает, что в качестве аргумента функция получает список,  заполненный целыми или вещественными числами. Результат работы функции вещественное число.

***❗❗❗ PEP-8! Код до и после функции отделяется двумя пустыми строками ***

	Можно  указать несколько типов через вертикальную черту

```python
a: int | float = 42  
b: float = 3,14  
a = a / b
```
